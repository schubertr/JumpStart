## JumpStart

Custom script generator that auto installs development software for OSX that you select. Similar to homebrew, but automated.

* Homebrew
* Homebrew Versions
* Homebrew Bundle
* Homebrew Services
* Homebrew Cask
* XCode
* OS X software updates

--Must have Homebrew installed for these--

* Atom
* VirtualBox
* Android Studio
* Sequel Pro
* SourceTree
